# PRTS 剧情抓取器 (PRTS Story Scraper)

这是一个简单的 Node.js 工具，用于从 PRTS Wiki 抓取明日方舟活动的剧情文本。它能自动过滤掉游戏指令和无关内容，只保留纯净的对话和旁白。

## 功能特点

- **自动搜索**：根据活动名称自动查找相关剧情页面。
- **全面覆盖**：支持抓取行动前 (BEG)、行动后 (END) 以及纯剧情 (ST/NBT) 页面。
- **智能过滤**：自动去除 AVG 指令、代码片段、UI 文本等干扰信息，生成适合阅读的文本。
- **简单易用**：支持交互式输入或命令行参数。

## 安装说明

1. 确保你的电脑上已安装 [Node.js](https://nodejs.org/)。
2. 下载本项目代码。
3. 在项目文件夹内打开终端（命令行），运行以下命令安装依赖：

```bash
npm install
```

## 使用方法

### 方式一：交互式运行（推荐）

直接运行脚本，程序会提示你输入活动名称：

```bash
npm start
```
或者：
```bash
node index.js
```

### 方式二：命令行参数

你也可以直接在命令后面加上活动名称：

```bash
node index.js "众生行记"
```

### Windows 用户

可以直接双击运行目录下的 `run.cmd` 脚本。

## 输出结果

程序运行完成后，会在当前目录下生成一个以活动名称命名的 `.txt` 文件（例如 `众生行记.txt`），其中包含了该活动的所有剧情文本。
